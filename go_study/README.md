### shortcut commands

To run some go file
```
go run file.go
```

To create go.mod file with your dependencies.
```
go mod init modulo
```

To build the project and create the file with name like inside go.mod 
```
go build
```

To install a external dependencie
```
go get github.com/badoux/checkmail
```
